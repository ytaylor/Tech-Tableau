### Generar paletas de colores con IA: 

- https://coolors.co/ (Página para generar paletas de colores)
- https://www.canva.com/ai

**Promp para generar Paleta de colores con una IA:**
- Crea una paleta de colores profesional inspirada en la identidad visual de Airbnb para usar en dashboards ejecutivos. La paleta debe incluir:
    - Un color principal basados en la marca Airbnb (Rojo Airbnb — #FF5A5F).
    - 6–8 colores secundarios coherentes con el look moderno, minimalista y premium de Airbnb (turquesas, grises suaves, tonos cálidos).
    - 3 colores de acento para alertas, KPI críticos y elementos destacados.
    - Una escala de gradiente para mapas de calor (desde gris claro hasta rojo Airbnb o turquesa Airbnb).
    - Variantes accesibles para usuarios con daltonismo, manteniendo contraste AA/AAA.
    - Estilo corporativo moderno: limpio, ligero, con colores suaves y saturación controlada.
    - Entregar todos los colores como códigos HEX listos para usar en Tableau, Power BI.
